Data Management and Visualization Practical
The Aim of all the experiments are as follows:
Lab 01:To draw a line chart where the inputs are already provided.
Lab 02a: To draw a histogram where the inputs are already provided.
Lab 02b: To draw a histogram where the inputs are provided by users.
Lab 03: To draw a line chart where the inputs are provided by the user.
Lab 04: To draw a moving circle which is animated.
Lab 05: To draw a moving line which is animated.
Lab 06: To draw a complete graph with vertices user defined and greater than 3.
Lab 07a: To draw a pie chart where input is given.
Lab 07b: To draw a pie chart where user will give the input.
Lab 08a: To draw a bar graph where input is given.
Lab 08b: To draw a bar graph where user will give the input.
Lab 09a: To draw a scatter plot where input is given.
Lab 09b: To draw a scatter plot where user will give the input.
Lab 10: To draw a subplot where bar graph and scatter plot are shown.
Lab 11: To write a Python code for handling missing variable.
Lab 12: To write a Python code for an animated circle using user input for frame size and speed.
Lab 13: To write a Python program to handle missing values, detect outliers, and visualize data using:
(i) bar chart
(ii) histogram
(iii) line chart
(iv) scatter plot
(v) pie chart.
Lab 14: To write a Python program to handle missing values and plot boxplot.
Lab 15: To write a Python program to create a scatter plot using dataset that may contain outliers, clusters, and positive/negative correlation.
