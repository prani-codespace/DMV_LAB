# Data Management and Visualization Practical

## Overview

This repository contains all lab assignments for the **Data Management and Visualization Practical** course.
Each experiment focuses on different data visualization techniques and data preprocessing concepts using Python.

---

## List of Experiments

### Lab 01 : To draw a line chart where the inputs are already provided.
### Lab 02a: To draw a histogram where the inputs are already provided.
### Lab 02b: To draw a histogram where the inputs are provided by the user.
### Lab 03: To draw a line chart where the inputs are provided by the user.
### Lab 04: To draw a moving circle using animation.
### Lab 05: To draw a moving line using animation.
### Lab 06: To draw a complete graph with user-defined vertices (greater than 3).
### Lab 07a: To draw a pie chart with given input.
### Lab 07b: To draw a pie chart using user input.
### Lab 08a: To draw a bar graph with given input.
### Lab 08b: To draw a bar graph using user input.
### Lab 09a: To draw a scatter plot with given input.
### Lab 09b: To draw a scatter plot using user input.
### Lab 10: To create a subplot displaying both a bar graph and a scatter plot.
### Lab 11: To write a Python program for handling missing values.
### Lab 12: To create an animated circle using user-defined frame size and speed.
### Lab 13: To write a Python program to:
* Handle missing values
* Detect outliers
* Visualize data using:
  * Bar Chart
  * Histogram
  * Line Chart
  * Scatter Plot
  * Pie Chart
### Lab 14: To handle missing values and plot a boxplot.
### Lab 15: To create a scatter plot using a dataset containing: Outliers, Clusters, Positive/Negative correlation
